# Try to learning .Net Core with C#
REST API with .Net Core

### Pipeline of process to develop
* Pull code => Git
* Build => dotnet build
* Unit Test => dotnet test --logger trx
* Code coverage (Soon)
* Deploy (Soon)
* API Testing (Soon)

### Step to run
```
$dotnet build
$dotnet test --logger trx
$dotnet run --project restapi/restapi.csproj
```
